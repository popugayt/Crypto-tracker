# crypto_tracker
## Рудаков Егор  Шашито Данил 9-3-ПО-22

## CryptoTracker это мобильное приложение для отслеживания курсов криптовалют  в реальном времени. 
## Пользователь может просматривать актуальные значения, динамику изменения курсов, графики за выбранный период и статистику. 
## Приложение использует открытые API, имеет современный тёмный интерфейс и удобную навигацию между экранами.
